WeGuess PHP API
===============


get_did_info
------------


accepts parameters:


(did=device_unique_id)


```javascript
"http://weguess.mobi/api.php?f=get_did_info&did=01234_example_device_id_56789"
```

returns JSON:

> {"email":"example@gmail.com","did":"01234_example_device_id_56789","imei":"1_1","categories":[1,3,7]}
